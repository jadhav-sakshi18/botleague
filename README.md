# BotLeague – Robotics Competition Platform

A responsive React + Vite web application for **BotLeague**, India's robotics competition platform. The project showcases events, rankings, competition categories, sponsors, and the robotics ecosystem with a modern gaming-inspired UI.

---

## Features

- Responsive landing page
- Hero section
- Competitions & Events
- Why Register section
- Competition Categories
- Competition Disciplines
- Leaderboard
- Sponsors section
- Join the Ecosystem
- Modern Footer
- Smooth scrolling navigation
- Reusable React components

---

## Tech Stack

- React
- Vite
- JavaScript (ES6)
- Tailwind CSS
- React Router DOM
- Lucide React Icons

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project folder

```bash
cd botleague
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

The application will be available at

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
